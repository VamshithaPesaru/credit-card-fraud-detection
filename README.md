# credit-card-fraud-detection
Credit card fraud detection project using Python and machine learning. Performs data exploration, visualization, class imbalance analysis, random sampling, feature encoding with OneHotEncoder, train-test split, and Logistic Regression model training to identify fraudulent transactions in real-world financial data.
### Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

### Model
- Logistic Regression
